# Contributing to Zedny Smart Course Recommender

Thank you for your interest in contributing! Please follow these guidelines to ensure a smooth workflow and high-quality codebase.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Write clear, production-grade code**. Follow PEP8 and use type hints.
3. **Add or update tests** for any new features or bug fixes.
4. **Document your changes** in the code and update the README if needed.
5. **Submit a pull request** with a clear description of your changes and why they are needed.

## Code Style
- Use Python 3.11 syntax and features only.
- Use [Black](https://black.readthedocs.io/en/stable/) for formatting.
- All functions and classes must have docstrings.

## Commit Messages
- Use clear, descriptive commit messages.
- Reference issues where applicable (e.g., `Fixes #42`).

## Reporting Issues
- Use the issue tracker to report bugs or request features.
- Provide as much detail as possible, including error messages and steps to reproduce.

## Code of Conduct
- Be respectful and inclusive.
- No harassment or discrimination will be tolerated.

---

Thank you for helping make this project better!
